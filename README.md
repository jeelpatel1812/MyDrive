### Configure Environment Variables
```

  PORT = 'port'
  CORS_ORIGIN = 'cors-origin'

  AWS_SECRET_KEY = 'aws-secret-key'
```

### Start or run project from both client and server
```bash
cd client
npm start

cd server
npm start
```
